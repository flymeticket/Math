Upload Spaces
=============

Public website files are sorted into these folders:

- `resources/pdfs/myp/` for MYP command-term sheets, criterion samples, guides, and other downloadable PDFs.
- `resources/pdfs/ai-sl/` for AI SL question papers and resources.
- `resources/pdfs/ai-hl/` for AI HL question papers, mark schemes, and resources.
- `resources/pdfs/aa-sl/` for AA SL question papers and resources.
- `resources/pdfs/aa-hl/` for AA HL question papers, mark schemes, and resources.
- `testimonials/photos/` for WhatsApp screenshots, parent messages, result photos, and other testimonial images.
- `testimonials/videos/` for testimonial videos.
- `_duplicates/` for duplicate files that should be kept but not shown on the website.

Recommended future file names:

- `myp-command-terms.pdf`
- `criterion-a-c-sample.pdf`
- `criterion-b-c-investigation-guide.pdf`
- `criterion-d-c-sample.pdf`
- `student-success-1.jpg`
- `student-success-1.mp4`

The website can serve files from this folder using URLs like:

- `/uploads/resources/pdfs/myp/myp-command-terms.pdf`
- `/uploads/resources/pdfs/aa-hl/aahl-paper-1-question-paper.pdf`
- `/uploads/testimonials/photos/student-success-1.jpg`
- `/uploads/testimonials/videos/student-success-1.mp4`
