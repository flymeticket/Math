Testimonial Media
=================

Paste testimonial files here:

- `photos/` for WhatsApp screenshots, result images, and student/parent message screenshots.
- `videos/` for testimonial videos.

Recommended names:

- `myp-student-success-1.jpg`
- `myp-parent-whatsapp-1.png`
- `myp-video-testimonial-1.mp4`

MYP pages will list files from these folders.
